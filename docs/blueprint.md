# **App Name**: Code Pulse

## Core Features:

- Problem Browser: Browse a curated list of coding challenges, filterable by topic and difficulty.
- Problem Details: View detailed problem descriptions, constraints, and example test cases.
- Code Editor: Write and edit code directly in the browser, with syntax highlighting and basic code completion. User is able to compile and run code for common programming languages
- Test Cases & Solutions: Display example test cases and verified solutions to check against the editor window.
- Personalized Challenges: AI powered feature to analyze the problems and give you relevant suggestions using a 'tool'.
- Course Catalog: Page featuring educational resources related to coding challenges.
- User Reviews and Projects: Browse and rate community coding challenges. Additionally, access code sandboxes to rate working projects.

## Style Guidelines:

- Background color: Dark background (#121212) to reduce eye strain during long coding sessions.
- Primary color: Vibrant purple (#A020F0) for a modern and energetic feel.
- Accent color: Bright green (#00FF00) for key actions and highlights, providing contrast and clarity.
- Body and headline font: 'Inter' sans-serif for clean readability in both the code editor and interface. 
- Code font: 'Source Code Pro' monospace for optimal code clarity.
- Minimalist icons with a focus on clarity and function.
- Subtle transitions and animations to enhance the user experience without being distracting.
